# Tetris.xdc

Simple HTML5-Canvas/JavaScript Tetris game implementation for Webxdc.

## Building

```
pnpm i
pnpm build
```

A `.xdc` file will be generated in the `dist` folder, then just send it in any chat
and you should be able to play the game.
